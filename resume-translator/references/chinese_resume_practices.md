# Chinese Resume Best Practices (中文简历)

## Overview

Chinese resumes (简历/履歷) follow conventions that differ between Mainland China (Simplified Chinese), Taiwan (Traditional Chinese), and Hong Kong (Traditional Chinese/English). This guide covers practices common across Chinese-speaking regions with notes on regional variations.

## Document Types

### 简历 (Jiǎnlì) / 履歷 (Lǚlì) - Resume/CV
- Comprehensive document covering personal details, education, and experience
- Length: 1-2 pages for entry to mid-level, up to 3 pages for senior positions
- Format: A4 size, can be template-based or custom
- Both PDF and Word formats acceptable (PDF preferred for final submission)

### Supporting Documents
- **求职信 (Qiúzhíxìn) / 自薦信** - Cover letter (sometimes requested)
- **作品集 (Zuòpǐnjí) / 個人作品** - Portfolio (for creative/technical roles)

## Standard Sections

### 1. Personal Information (个人信息 / 個人資料)

**Essential Information:**

**Mainland China (简体中文):**
- Name (姓名): Full name in Chinese characters
- Gender (性别): 男 (male) / 女 (female)
- Date of birth (出生日期): YYYY年MM月 or YYYY-MM-DD
- Phone (电话/手机): Mobile number with country code if applicable
- Email (邮箱/电子邮件): Professional email address
- Current location (现居地): City and province
- Photo (照片): Professional headshot (3×4cm or 2×3cm)

**Taiwan (繁體中文):**
- Name (姓名): Chinese characters, may include English name
- Gender (性別): 男 / 女
- Birth date (出生日期): ROC calendar or Western calendar
- Phone (電話): Mobile and/or home number
- Email (電子郵件): Professional email
- Address (地址): May include full address
- Photo (照片): Professional photo

**Hong Kong:**
- Similar to Taiwan but often bilingual (Chinese/English)
- May include both Chinese and English names prominently
- Address and contact details

**Optional but Common:**
- Political status (政治面貌): 党员 (party member), 群众 (non-party) - Mainly in Mainland China
- Marital status (婚姻状况): 未婚 (single), 已婚 (married) - Less common now
- Nationality (国籍/籍贯): If relevant
- Hukou/Household registration (户口): Mainland China only
- ID number: Generally NOT included due to privacy concerns

**Photo Guidelines:**
- Professional business attire
- Neutral background (white, light blue, or light gray)
- Clear facial features, natural expression
- Recent photo (within 6 months)
- Standard in Mainland China and Taiwan; optional in Hong Kong for international companies

### 2. Job Objective (求职意向 / 應聘職位)

Brief statement of target position:
- Desired position (意向职位): Specific job title
- Industry (期望行业): Target industry sector
- Location (工作地点): Preferred work location
- Salary expectations (期望薪资): Optional, often discussed later

**Example (Simplified):**
```
求职意向:软件工程师 | 互联网/IT行业 | 北京 | 月薪面议
```

**Example (Traditional):**
```
應聘職位:軟體工程師 | 網路/資訊科技產業 | 台北
```

Note: This section is more common in Mainland China; may be omitted in Taiwan/Hong Kong where cover letters serve this purpose.

### 3. Educational Background (教育背景 / 教育經歷)

**Format:** Reverse chronological order (most recent first)

**Information to include:**
- Time period: YYYY.MM - YYYY.MM
- Institution name (学校/院校): Full official name
- Degree (学历): 本科/学士 (bachelor's), 硕士 (master's), 博士 (doctorate)
- Major/Specialization (专业): Field of study
- GPA (学分绩点): If strong (3.5+/4.0 or 85+/100)
- Class ranking (排名): If in top percentage
- Relevant coursework (主修课程): For recent graduates

**Example (Simplified):**
```
2015.09 - 2019.06  清华大学  计算机科学与技术学士  GPA: 3.8/4.0
                   主修课程:数据结构、算法设计、机器学习
                   荣誉:校级一等奖学金(2017、2018)
```

**Example (Traditional):**
```
2015.09 - 2019.06  國立台灣大學  資訊工程學系學士  GPA: 3.8/4.0
                   主修課程:資料結構、演算法設計、機器學習
                   榮譽:書卷獎(2017、2018)
```

**Notes:**
- Include high school only for recent graduates or if particularly prestigious
- Study abroad experiences should be highlighted
- Include academic honors, scholarships, and awards

### 4. Work Experience (工作经历 / 工作經驗)

**Format:** Reverse chronological order

**For each position:**

**Company Information:**
- Time period: YYYY.MM - YYYY.MM (or 至今 for current)
- Company name (公司名称): Full name
- Industry/Sector (行业): Brief company description if not well-known
- Position/Title (职位): Your role
- Department (部门): If relevant

**Responsibilities and Achievements:**
- Use bullet points for clarity
- Start with action verbs
- Quantify achievements with specific metrics
- Highlight leadership and initiative
- Include project details for technical roles

**Example (Simplified):**
```
2020.07 - 至今     腾讯科技有限公司  |  互联网
职位:高级软件工程师  |  部门:微信事业群

工作职责与业绩:
• 负责微信支付功能的后端开发与维护,服务日活用户超过5亿
• 主导交易系统性能优化,将响应时间降低40%,提升用户体验
• 带领5人技术团队完成跨境支付项目,按时交付并获得季度优秀团队奖
• 参与技术分享,撰写内部技术文档10余篇
```

**Example (Traditional):**
```
2020.07 - 至今     台積電股份有限公司  |  半導體製造
職位:資深製程工程師  |  部門:先進製程開發部

工作職責與成果:
• 負責7nm製程良率提升項目,使良率從85%提升至92%
• 主導新製程參數優化,縮短生產週期15%
• 跨部門協作解決20餘項技術難題,確保量產時程
• 培訓新進工程師,建立標準作業程序文件
```

### 5. Skills (技能专长 / 專業技能)

**Technical Skills (专业技能):**
- Programming languages: List with proficiency level
- Software/Tools: Relevant applications
- Frameworks and technologies
- Use levels: 精通 (proficient), 熟练 (skilled), 了解 (familiar)

**Example:**
```
编程语言:Python(精通), Java(熟练), C++(了解)
开发框架:Django, Spring Boot, React
工具平台:Git, Docker, AWS, MySQL
```

**Language Proficiency (语言能力 / 語言能力):**

**Mainland China:**
- English: CET-4, CET-6, IELTS, TOEFL, or descriptive (fluent, business level)
- Other languages: HSK for Chinese learners, indicate proficiency level

**Taiwan:**
- English: TOEIC, TOEFL, IELTS scores or GEPT level
- Other languages with proficiency description

**Example:**
```
英语:CET-6(580分),TOEFL(105分),听说读写能力良好
日语:JLPT N2,可进行日常会话
```

**Certifications (资格证书 / 證照):**
- Certificate name
- Issuing organization
- Date obtained
- Certificate number (optional)

### 6. Project Experience (项目经历 / 專案經驗)

Especially important for technical roles and recent graduates:

**For each project:**
- Project name (项目名称)
- Time period: YYYY.MM - YYYY.MM
- Project description (项目描述): Brief overview
- Your role (担任角色): Your specific responsibilities
- Technologies used (使用技术): Tech stack
- Outcomes (项目成果): Results and achievements

**Example:**
```
电商平台后台管理系统  |  2022.06 - 2022.12
项目描述:为中型电商企业开发全栈管理系统,包含商品管理、订单处理、数据分析等模块
担任角色:全栈开发工程师,负责后端API设计与前端界面开发
使用技术:Vue.js, Node.js, MongoDB, RESTful API
项目成果:系统上线后处理效率提升60%,获得客户高度认可,项目在团队中被评为优秀案例
```

### 7. Campus Activities / Internships (校园活动 / 实习经历)

For recent graduates or those with limited work experience:

**Internships (实习经历):**
- Similar format to work experience
- Emphasize skills learned and contributions made

**Campus Activities (校园活动):**
- Student organizations and leadership roles
- Competitions and awards
- Volunteer work
- Research projects

### 8. Honors and Awards (荣誉奖励 / 獲獎經歷)

- Date: YYYY.MM
- Award name (奖项名称): Official title
- Issuing organization (颁发机构)
- Brief description if not self-explanatory

**Example:**
```
2023.06  腾讯优秀员工奖  |  腾讯科技有限公司
2022.11  全国软件设计大赛一等奖  |  中国计算机学会
2021.12  优秀毕业生  |  清华大学
```

### 9. Self-Assessment (自我评价 / 個人特質)

**⚠️ IMPORTANT for Technical Roles (AI/ML, Engineering, Data Science):**
- **MINIMIZE or SKIP this section** - Modern technical resumes globally prioritize concrete achievements over self-descriptions
- If cultural expectations require it, limit to 1-2 sentences about:
  - Technical domain expertise (e.g., "5年机器学习工程经验" / "5年機器學習工程經驗")
  - Scale of work (e.g., "服务日活用户超过1亿" / "服務日活用戶超過1億")
- **AVOID generic phrases** like:
  - "工作严谨认真" (work conscientiously)
  - "具备较强的学习能力" (strong learning ability)
  - "期待职业成长" (career growth aspirations)
- These statements provide no verifiable value and waste precious space
- Focus on quantifiable achievements in the Work Experience section instead

**For Non-Technical Roles:**
Optional brief section (1-2 sentences, shorter than traditional):
- Specific quantifiable achievements that demonstrate strengths
- Domain expertise relevant to target position
- Avoid generic personality traits

**If required by company culture, use this pattern (Technical roles):**
```
Simplified: 具有5年大规模机器学习系统开发经验,专注于NLP与语音处理。曾主导服务日活5000万用户的生产系统,实现模型推理延迟降低40%、准确率提升15%。
Traditional: 具有5年大規模機器學習系統開發經驗,專注於NLP與語音處理。曾主導服務日活5000萬用戶的生產系統,實現模型推理延遲降低40%、準確率提升15%。
```

**Traditional example to AVOID:**
```
❌ 具有5年互联网产品开发经验,擅长团队协作与跨部门沟通。工作严谨认真,注重代码质量与用户体验。
具备较强的学习能力与问题解决能力,能够快速适应新技术与业务场景。期待在更具挑战的环境中实现职业成长。
(Too generic, no metrics, wastes space that could show real achievements)
```

## Formatting Guidelines

### Layout and Design
- Clean, professional appearance
- Consistent formatting throughout
- Clear section headings
- Adequate white space (not too cramped)
- Use of tables or structured layouts acceptable
- Conservative color scheme (black, navy, dark gray)

### Font and Typography

**Mainland China (Simplified):**
- Font: 宋体 (SimSun), 微软雅黑 (Microsoft YaHei), 黑体 (SimHei)
- Size: 10-11pt body, 12-14pt headings, 9pt for fine details
- Emphasis: Bold for headings and key information

**Taiwan (Traditional):**
- Font: 新細明體 (PMingLiU), 微軟正黑體 (Microsoft JhengHei), 標楷體 (DFKai-SB)
- Size: Similar to Simplified
- Clear hierarchy with font weight and size

**General:**
- Left-aligned text preferred
- Use bullet points for lists
- Maintain consistent spacing
- Tables for structured information (personal details, education)

### Language and Tone
- Use formal, professional Chinese (书面语)
- Be concise and direct
- Use appropriate business terminology
- Avoid overly flowery language
- Balance modesty with confidence
- Use active voice and action verbs

## Regional Variations

### Mainland China (大陆)
- More detailed personal information traditionally included
- Political status may be relevant for government/SOE positions
- Emphasis on collective achievements and team contributions
- Party membership noted if applicable
- Hukou (household registration) for certain positions

### Taiwan (台湾)
- Somewhat less personal information
- ROC calendar may be used alongside Western calendar
- Military service status for males (relevant for certain ages)
- More acceptance of English resumes for international companies
- Emphasis on education and continuous learning

### Hong Kong (香港)
- Often bilingual (Chinese and English)
- Less personal information, closer to international standards
- Strong emphasis on language skills (Cantonese, Mandarin, English)
- International experience highly valued
- More concise format, typically 1-2 pages

## Common Mistakes to Avoid

1. **Poor quality or unprofessional photo**: Use recent, appropriate business photo
2. **Too much personal information**: Avoid ID numbers, detailed family information
3. **Inconsistent formatting**: Maintain uniform style, dates, spacing
4. **Vague descriptions**: Be specific with metrics and achievements
5. **Too lengthy**: Keep to 2-3 pages maximum unless very senior
6. **Typos and errors**: Proofread carefully; errors suggest carelessness
7. **Overly decorative design**: Keep professional, avoid excessive colors/graphics
8. **Mixing simplified and traditional characters**: Use one system consistently
9. **Generic content**: Customize for each application
10. **Omitting contact information**: Ensure phone and email are current and professional

## Date Formatting

**Mainland China:**
- Standard: YYYY年MM月 or YYYY.MM or YYYY-MM
- Example: 2020年01月 or 2020.01 or 2020-01

**Taiwan:**
- ROC calendar: YYY年MM月 (where YYY = Western year - 1911)
- Western calendar: YYYY年MM月 or YYYY.MM
- Example: 109年01月 (ROC) = 2020年01月 (Western)

**Hong Kong:**
- Typically Western calendar: YYYY.MM or MM/YYYY
- Example: 2020.01 or 01/2020

**Period notation:**
- Use – or ~ for ranges
- Example: 2020.01 – 2023.05 or 2020年1月~2023年5月
- Present: 至今 (Mainland/Taiwan) or 現在 (Hong Kong)

## File Naming Convention

**Mainland China (Simplified):**
- 姓名_应聘职位_简历.pdf or 简历_姓名_YYYYMMDD.pdf
- Example: 张伟_软件工程师_简历.pdf or 简历_张伟_20231115.pdf

**Taiwan (Traditional):**
- 姓名_履歷.pdf or 履歷_應徵職位_姓名.pdf
- Example: 陳小明_履歷.pdf or 履歷_軟體工程師_陳小明.pdf

**For international companies:**
- Resume_Name_YYYYMMDD.pdf
- Example: Resume_ZhangWei_20231115.pdf

## Tips for Resume Translation

### English to Chinese:
- Add personal information section with photo
- Adapt structure to emphasize education (typically more important in Chinese context)
- Use Chinese business terminology and job titles
- Include more detailed project descriptions
- Add self-assessment section
- Consider regional variations (Simplified vs. Traditional)
- Adapt metrics to local context when possible

### Chinese to English:
- Remove or minimize personal details (photo, age, marital status, etc.)
- Condense to 1-2 pages
- Emphasize achievements and impact with quantifiable results
- Use action verbs at the start of bullet points
- Focus on transferable skills and international experience
- Remove political/hukou information
- Adapt to Western resume conventions (more direct self-promotion)

## Cultural Considerations

### Education and Credentials
- Educational background carries significant weight
- University name and ranking are important
- Academic honors should be prominently displayed
- Continuous learning and certifications valued

### Work Experience
- Loyalty vs. mobility varies by generation and industry
- Traditional industries may value longer tenures
- Tech and multinational companies more accepting of job changes
- Frame career moves positively: growth, learning, challenges

### Modesty vs. Self-Promotion
- Balance Chinese cultural value of modesty with need to showcase abilities
- Use "we" for team achievements but clarify individual contributions
- Be factual and specific rather than boastful
- Let achievements speak for themselves

### Relationships and Networks
- Alumni networks (校友) are influential
- Referrals carry weight
- Professional connections matter
- Consider mentioning shared connections if appropriate

### Age and Experience
- Career progression relative to age is evaluated
- Gaps in employment should be explained
- Continuous employment history preferred
- Recent experience most relevant
